那么本章是基于redis作为存储中间件，实现服务治理，也就是图片中的第1，2，3步，思路跟zookeeper实现方式一致，存储结构也大致相同。使用redis的list类型。想详细了解redis的，请跳转至redis妙用，这里讲述了redis的各种应用场景，以及将遇到的坑。
